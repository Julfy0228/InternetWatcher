# InternetWatcher

InternetWatcher — небольшая утилита для Windows, которая отображает состояние интернет‑подключения прямо в системном трее.  
Приложение периодически проверяет несколько серверов, измеряет пинг и визуализирует результат в виде мини‑гистограммы из 8 столбиков (иконка 16×16).

Каждый столбик соответствует равномерно распределённой группе серверов.  
Цвет и высота столбика показывают качество соединения:

- 🟩 **Зелёный (полная высота)** — быстрый отклик  
- 🟨 **Жёлтый (2/3)** — медленный отклик  
- 🟥 **Красный (1/3)** — сервер недоступен  

В контекстном меню отображается подробный статус каждого сервера: HTTP‑код и пинг.

## Возможности

- Проверка доступности нескольких серверов (HTTP 204)
- Измерение пинга
- Равномерное распределение серверов по столбикам
- Динамическая иконка в системном трее (16×16)
- Цветовая индикация состояния сети
- Меню со статусом каждого сервера
- Минимальная нагрузка на систему
